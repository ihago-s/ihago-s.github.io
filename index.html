Node.js学习（1）-基础知识

发表于 2018-07-25 | 分类于 Notes | | 阅读次数:

Node.js是什么？官网给出的解释是：
Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。Node.js 的包管理器 npm，是全球最大的开源库生态系统。

Node.js中的模块

  • assert- 断言，assert模块提供了断言测试的函数，用于测试不变式
  • Buffer- 缓存，Buffer类在Node.js中是一个全局变量，用于实现二进制数据的存储和转换
  • child_process- 子进程，提供衍生子进程的功能
  • cluster- 集群，用于实现多进程
  • console- 控制台，简单的调试控制台，类似于Web浏览器提供的JavaScript控制台
  • crypto- 加密，提供加密功能，包含对 OpenSSL 的哈希、HMAC、加密、解密、签名、以及验证功能的一整套封装
  • dgram- 数据报，提供UDP数据包scoket的实现
  • dns- 域名服务器，用于实现与dns相关的处理
  • Error- 错误，提供Node.js引起的javaScript错误和系统错误
  • events- 事件，用于为事件处理提供一个基础类
  • fs- 文件系统，用于操作文件和文件系统
  • http- HTTP，用于实现HTTP服务器端及客户端
  • https- HTTPS，用于实现HTTPS服务器端及客户端
  • net- 网络，用于创建TCP服务器与客户端
  • os- 操作系统，提供了一些操作系统相关的实用方法
  • path- 路径，提供了一些用于处理文件和目录路径的工具函数
  • querystring- 查询字符串，用于处理HTTP请求中使用的查询字符串
  • readline- 逐行读取，用于读取一行标准输入流
  • repl- 交互式解释器，用于实现REPL（Read-Eval-Print-Loop）交互式运行环境
  • stream- 流，用于构建实现了流接口的对象
  • string_decoder- 字符串解码器，用于从二进制数据到字符串数据之间的转换
